Glisse un modèle .glb (ex. sprites/kenney_racing-kit/Models/GLB format/raceCarRed.glb), règle, puis télécharge la sprite sheet. Nécessite une connexion (three.js via CDN).

Aperçu : le modèle tourne. Si le nez ne pointe pas « vers le haut » à l'angle 0, ajuste « Rotation frame 0 », ou bien plus tard dir/base dans le jeu.
